# Assets

This folder is reserved for optional local demo assets.

The public demo workflow uses Wikimedia Commons `Special:FilePath` image URLs for Rider-Waite-Smith tarot cards instead of storing card images directly in the repository.

Example:

```text
https://commons.wikimedia.org/wiki/Special:FilePath/RWS_Tarot_19_Sun.jpg
```

## Important

Do not upload card images unless you are sure the license allows public redistribution.

For portfolio usage, public image URLs are safer than storing image files directly.

Before a demo, verify that each referenced image URL still resolves successfully.
