# Maintenance Notes

This file keeps lightweight maintenance rules for the Daily Tarot Card Bot repository.

## Regular Checks

- Keep Telegram token examples as placeholders only.
- Keep demo screenshots free from private chats.
- Review scheduled-message behavior after workflow changes.
- Document user privacy assumptions.
- Update project notes when product scope changes.

## Safe Data Rules

Use placeholders for tokens, chat IDs, user IDs, private messages, n8n credentials, webhook URLs, database passwords, and subscriber exports.

## Release Hygiene

Before publishing a new update, check README accuracy, security notes, setup instructions, demo data, and legal disclaimer.
