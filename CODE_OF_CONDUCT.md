# Code of Conduct

## Our Standard

This repository is maintained as a public automation and portfolio project. Discussions should be respectful, practical, and focused on improving the project.

Examples of good behavior:

- clear issue reports;
- constructive technical feedback;
- respectful product discussion;
- safe examples without private Telegram data;
- useful documentation and automation suggestions.

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or personal attacks;
- publishing Telegram tokens, chat IDs, user IDs, private messages, or credentials;
- spam;
- abusive language;
- irrelevant promotion;
- misleading claims about predictions or outcomes.

## Enforcement

Maintainers may edit, close, delete, or lock issues and pull requests that violate this code of conduct.

## Reporting

Report problems through GitHub issues with a safe, factual description. Do not include private messages, credentials, tokens, or personal information.
